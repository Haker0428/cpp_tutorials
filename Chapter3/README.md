# 第三章 处理数据

### 1. 关于整型字面值
C++使用前一(两)位来标识数字常量的基数。**如果第一位为1~9，则基数为10(十进制)；因此93是以10为基数的。如果第一位是0，第二位是1~7，则基数为8(八进制);因此042的基数是8，它相当于十进制的34。如果前两位为0x或0X,则基数为16(十六进制),因此0x42为十六进制数，相当于十进制数66**
